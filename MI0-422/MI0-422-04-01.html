(3)인공지능 모델 개발

우리 반 친구들의 얼굴을 인식하여 자동으로 출석을 확인하는 인공지능 모델을 만들어 보자.

모델 학습

아래의 내용을 바탕으로 모둠별 학생 수만큼 클래스를 나누고 각자의 얼굴을 촬영하여 데이터를 수집하고 학습하여 모델을 만든다.

모델의 종류, 학습 모델 이름, 클래스 이름, 수집 데이터, 수집 방법을 정리한 표
모델의 종류 분류: 이미지
학습 모델 이름 우리 반 얼굴 인식
클래스 이름
예

시아, 영인, 지영, 지민
수집 데이터 얼굴 촬영 이미지
수집 방법 얼굴을 직접 촬영

확인하기

1.인공지능 모델 학습에서 훈련 데이터와 테스트 데이터를 나눠 사용하였는가?

2.인공지능 모델에서 발생한 오류에 대해 수정과 보완을 진행하였는가?

결과 확인

새로운 데이터를 활용하여 완성된 모델의 학습 결과를 평가한다. 만약 분류가 잘 되지 않으면 다음처럼 해결한다.

더 많은 이미지 자료를 추가하여 학습한다.

기존의 사진 자료를 수정, 가공한다(단색 배경, 얼굴 크기나 각도가 다르게 촬영하기 등).
