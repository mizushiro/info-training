(3)소리 분류 모델: 노래 제목 인식하기

모델 학습

아래의 내용을 바탕으로 모델을 학습시킨다.

모델의 종류, 학습 모델 이름, 클래스 이름, 수집 데이터, 수집 방법을 정리한 표
모델의 종류 분류: 소리
학습 모델 이름 노래 인식 인공지능
클래스 이름 노래 제목들(예진도 아리랑, 밀양 아리랑, 정선 아리랑)
수집 데이터 각 클래스에 맞는 노래나 음악 소리 데이터
수집 방법 직접 노래를 녹음하거나 인터넷에서 음악 파일 수집하기

확인하기
