(1) 인공지능 모델 만들기

개와 고양이 이미지 데이터를 이용해 인공지능 모델을 만들어 보자.

데이터 입력: 모델 이름과 클래스별 데이터 입력

1 모델 이름 입력하기: 만들고자 하는 모델의 이름을 입력한다.

2 데이터 입력하기: 모델 학습에 필요한 데이터를 수집하여 클래스별로 분류하여 입력한다. 데이터는 미리 저장된 파일을 업로드하거나 카메라로 바로 촬영해 입력할 수 있다. 클래스별로 충분히 학습할 수 있도록 최소한 5개 이상의 데이터를 입력한다.

학습: 학습하여 모델 만들기

3 모델 학습하기: 클래스별로 입력한 데이터로 인공지능 시스템을 학습시켜 분류할 수 있는 모델을 만든다.

결과 확인: 모델 성능 검증하기

4 결과 확인하기: 학습된 모델이 정확하게 분류하는지 결과를 확인하고 평가한다. 이때 학습에 사용된 데이터를 사용하지 않고 새로운 데이터를 입력해야 한다. 학습된 모델은 새로운 데이터가 무엇인지 확률값으로 보여 준다.
